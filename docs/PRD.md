# Product Requirements Document (PRD)

## Product
GitHub Profile Analyzer v1

## Job To Be Done
Turn raw GitHub history into a credible developer identity with an evidence-grounded profile README.

## Personas
- Primary: students and early-career developers building public credibility.
- Secondary (future): recruiter-facing consumers of structured profile signals.

## v1 Features
- Username-based analysis run.
- Signal dashboard (consistency, ownership, polish, orientation).
- Engineering archetype with confidence + evidence.
- Activity timeline summary.
- Standout project surfacing.
- Evidence-backed README generation.
- Shareable report output.

## Non-Goals
- ATS integration.
- Candidate ranking and hiring decisions.
- Enterprise workflow features.

## Success Metrics
- Analysis completion rate.
- Share/report export rate.
- README publish/adoption rate.
- User-perceived accuracy score.
- Profile-edit conversion rate after report generation.
