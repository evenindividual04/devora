# Detailed Implementation Plan

## Phase 1: Foundation
- FastAPI app + route skeleton.
- Typed request/response contracts.
- In-memory orchestration baseline.

## Phase 2: Deterministic Analysis
- Repo/commit ingestion interfaces.
- Initial deterministic signals: consistency, ownership, polish, orientation.

## Phase 3: Semantic Layer
- Archetype classification and evidence ranking.
- Timeline generation primitives.

## Phase 4: Narrative Layer
- README/report synthesis constrained by evidence refs.
- Honesty mode controls in narrative output.

## Phase 5: Product UX (next)
- Analysis status stream, report explorer, README export, evidence drilldown.

## Phase 6: Shareability + Expansion Hooks (next)
- Snapshot/share endpoints.
- Recruiter-compatible read-only signal views.
